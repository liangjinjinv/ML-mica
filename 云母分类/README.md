# 云母分类系统 - 使用版

这是一个精简版的云母分类系统，专为实际使用而优化，删除了所有调试信息和日志打印，只保留核心功能。

## 特点

- **精简设计**：删除所有调试信息和日志打印，界面更清爽
- **核心功能**：保留完整的主量元素和微量元素预测功能
- **双模型支持**：支持主量元素模型（13个特征）和微量元素模型（19个特征）
- **批量处理**：支持批量预测功能
- **实时预测**：直接显示模型原始预测结果

## 文件结构


## 功能特性

### 预测模式
1. **单样本预测**：分别进行主量元素和微量元素预测
2. **双模型预测**：同时使用两个模型进行预测
3. **批量预测**：支持上传JSON文件进行批量处理

### 输入特征
- **主量元素**（13个）：SiO2, TiO2, Al2O3, FeO, MnO, MgO, CaO, Na2O, K2O, F, Cl, Mg#, A/CNK
- **微量元素**（19个）：Li, Sc, V, Cr, Co, Ni, Rb, Sr, Nb, Sn, Cs, Ba, Ta, W, Nb/Ta, V/Sc, Rb/Sr, Rb/Ba, Nb/Sn

### 输出类别
- Sn（锡型）
- W（钨型）  
- 斑岩Cu（斑岩铜型）

## 使用方法

1. **双击运行**：双击 `一键启动.bat` 文件
2. **自动启动**：系统会自动启动后端服务并打开前端界面
3. **输入数据**：在界面中输入相应的元素含量数据
4. **选择预测**：点击相应的预测按钮获取结果

## 系统要求

- Python 3.7+
- Windows操作系统

## 依赖包

运行前请确保安装以下依赖：
```
pip install -r requirements.txt
```

## API端点

- `GET /health` - 健康检查
- `POST /predict/major` - 主量元素预测
- `POST /predict/trace` - 微量元素预测
- `POST /predict/both` - 双模型预测
- `POST /predict/batch` - 批量预测

## 注意事项

- 确保模型文件存在，否则系统会使用模拟模型
- 输入数据时请确保所有必需特征都有值
- 预测结果直接显示模型原始输出，无额外处理
